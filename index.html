How to add a new roadmap

Four steps to extend this collection
1 Create data/your-topic-data.js — export TOPICS and PHASE_LABELS following the existing data file format.
2 Copy devops-sre.html to your-topic.html and change the <script src> to point to your new data file.
3 Add a new <a class="roadmap-card"> block in index.html before the placeholder card, following the pattern above.
4 Pick an accent color and matching tags. That's it — the shared renderer handles the rest automatically.
